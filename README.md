# StickyScroll

Unity-компонент для **бесконечного скролла** с пулом элементов (infinite recycling).

## Особенности

- Эффективный пул из 3 контейнеров (минимальное использование памяти и GC).
- Плавная инерция + drag (аналогично ScrollRect, но оптимизировано).
- Поддержка верхнего и нижнего лимита элементов.
- Простая расширяемость через `AbstractContentContainer`.
- Интеграция с VContainer (DI).
- Работает с любыми UI-элементами (UGUI).
